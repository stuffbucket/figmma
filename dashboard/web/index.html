figmma Setup

Configure your Figma connection to get started.

1

Personal Access Token

required
  1. Log in to Figma and open the file browser
  2. Click the account menu (top-left corner) → Settings
  3. Select the Security tab
  4. Under Personal access tokens, click "Generate new token"
  5. Set the expiration and scopes your token needs, then click Generate token
  6. Copy the token (you won't be able to see it again) and paste it below:
2

Team & Organization

optional
  1. Open Figma in your browser (log in if needed)
  2. In the left sidebar, click on a team name (e.g. “AI Projects”)
  3. The URL should look like:
    figma.com/files/{orgId}/team/{teamId}
     ⚠ A file URL (/design/...) or org-only URL (/files/{orgId}) won’t have a team ID.
  4. Paste the full URL below — the org and team IDs will be extracted automatically:
